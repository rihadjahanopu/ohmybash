## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎨 **100+ Themes** | Beautiful, customizable terminal themes including Powerline support |
| 🔌 **Rich Plugins** | Git, Docker, Node.js, Python, and many more productivity plugins |
| ⚡ **Auto-Updates** | Keep your setup fresh with automatic update notifications |
| 🛠️ **Smart Aliases** | Hundreds of useful aliases for common commands |
| 🔧 **Easy Customization** | Simple configuration through `~/.bashrc` |
| 🐧 **Cross-Platform** | Works on Linux, macOS, and WSL |

## 🚀 Quick Start

### Prerequisites
- Unix-like OS (Linux/macOS/WSL)
- `curl` or `wget` installed
- `git` installed
- Bash 3.2+ (Bash 4+ recommended)

### One-Line Installation

**Via curl:**
```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/rihadjahanopu/ohmybash/master/tools/install.sh)"
